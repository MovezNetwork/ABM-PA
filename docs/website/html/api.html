ABM API¶

These API describes the methods in the src folder. They are methods related to population, model, simulation and tuning.

Population (src/population.py)¶

class src.population.CommunicationDataPopulation(name, input_args)¶

Population Class build on Communication Data.

create_population(graph)¶

Create a communication-data population in a graph representation. The connections (edges) are created and weighted based on individual communication data. The agents (nodes) are assigned attributes regarding age, gender, PAL, env.

Parameters

graph (Graph) – initial empty graph

Returns: NetworkX DiGraph: updated graph with weighted connections.
